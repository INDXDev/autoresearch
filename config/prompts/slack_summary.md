Summarise the day's Auto Research report for a Slack channel.

Keep it short and scannable:

- One line per section (Research News, Hypotheses, Related Work).
- Lead with the single most interesting point.
- Use plain text with Slack-style `*bold*` for emphasis; no long paragraphs.
- Do not include any secrets, tokens, or webhook URLs.

Note: the default template ships with a self-contained Python summary that does
not require an LLM. This file is provided so you can customise the Slack format
(for example, by wiring it into an AI summarisation step later).
